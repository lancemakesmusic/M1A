# M1A Logo Assets

This folder contains solid versions of the M1A logo for marketing and branding purposes.

## Available Logo Files

### Full Color Versions
- **`m1a-logo-full-color.svg`** - Complete logo with gold hexagram and silver M (512x512px)
  - Best for: Web use, presentations, full-color printing
  - Format: SVG (scalable vector graphics)
  - Colors: Gold (#D4AF37) and Silver (#C0C0C0) gradients

### Icon Version
- **`m1a-logo-icon.svg`** - Simplified icon version (256x256px)
  - Best for: App icons, favicons, small displays
  - Format: SVG
  - Colors: Gold and Silver gradients

### Minimal Version
- **`m1a-logo-minimal.svg`** - Just the M letter (256x256px)
  - Best for: Minimalist branding, watermarks
  - Format: SVG
  - Colors: Gold gradient

### Single Color Versions
- **`m1a-logo-solid-black.svg`** - Solid black version
  - Best for: Single-color printing, black backgrounds
  - Format: SVG
  - Color: Black (#000000)

- **`m1a-logo-solid-white.svg`** - Solid white version
  - Best for: Dark backgrounds, reverse printing
  - Format: SVG
  - Color: White (#FFFFFF)

## Usage Guidelines

### Converting to PNG/JPG
SVG files can be converted to raster formats using:
- Online tools: CloudConvert, Convertio
- Design software: Adobe Illustrator, Inkscape, Figma
- Command line: ImageMagick, Inkscape CLI

### Recommended Sizes
- **Web/App Icons**: 256x256px, 512x512px
- **Social Media**: 1024x1024px (square), 1200x630px (Facebook)
- **Print**: 300 DPI minimum (1500x1500px for 5" print)

### Color Specifications
- **Primary Gold**: #D4AF37 (RGB: 212, 175, 55)
- **Light Gold**: #FFD700 (RGB: 255, 215, 0)
- **Dark Gold**: #B8860B (RGB: 184, 134, 11)
- **Primary Silver**: #C0C0C0 (RGB: 192, 192, 192)
- **Light Silver**: #F5F5F5 (RGB: 245, 245, 245)
- **Dark Silver**: #A8A8A8 (RGB: 168, 168, 168)

### Design Elements
- **Hexagram**: 6-pointed star (Star of David) pattern
- **Central M**: Geometric M letter representing "M1A"
- **Style**: Modern, geometric, premium aesthetic

## Brand Guidelines

1. **Minimum Size**: Logo should never be smaller than 24px in height
2. **Clear Space**: Maintain clear space equal to the height of the M letter around the logo
3. **Color Usage**: 
   - Use full-color version on light backgrounds
   - Use white version on dark backgrounds
   - Use black version for single-color printing
4. **Do Not**:
   - Stretch or distort the logo
   - Change colors (except to black/white for single-color use)
   - Add effects that obscure the design
   - Rotate the logo

## Technical Notes

- All SVG files are optimized for web and print use
- Files use viewBox for scalability
- Gradients are defined for smooth color transitions
- Files are compatible with all modern browsers and design software
