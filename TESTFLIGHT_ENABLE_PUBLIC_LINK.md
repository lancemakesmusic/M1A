## Option 1: Enable Public Link (Easiest - No Tester Management)

### Steps:
1. **Click on "EXTERNAL TESTING"** in the left sidebar (under Testers)
2. **Click the blue "+" button** next to "EXTERNAL TESTING"
3. **Create a new External Testing Group:**
   - Name it (e.g., "Public Testing" or "Beta Testers")
   - Click "Create"
4. **Add Build #17 to the group:**
   - Select build 1.0.0 (17)
   - Click "Next"
5. **Enable Public Link:**
   - Look for "Public Link" toggle or section
   - Enable it
   - Copy the link (format: `https://testflight.apple.com/join/XXXXXX`)

## Troubleshooting

### "No builds available for testing"
- Make sure build #17 is added to a testing group
- Check that the build status is "Complete"

### "Public link not available"
- First-time external testing requires App Store review
- Wait 24-48 hours after adding build to external testing
- Check App Store Connect for review status

### Build Expired
- TestFlight builds expire after 90 days
- Submit a new build if needed
